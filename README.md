# SocialHub

AI-powered social media operating system — create once, optimize automatically, publish everywhere.

## Repository Structure

```
socialhub/
├── frontend/   # Flutter Web application (PWA)
├── backend/    # NestJS API
├── docs/       # Architecture, blueprint, and process documentation
└── .github/    # CI/CD workflow definitions
```

## Status

Project scaffold only. Application code is being built incrementally following the
implementation blueprint in `docs/blueprint/` (added in a later milestone).

## Quickstart

Coming online as Phase 0 milestones land:
- Milestone 0.2 — Flutter Web app shell
- Milestone 0.3 — NestJS API skeleton
- Milestone 0.4 — Database provisioning

Once those are in place, local setup will be a single `docker-compose up`.
